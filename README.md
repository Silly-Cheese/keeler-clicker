# Keeler Clicker

Keeler Clicker is a goofy, cartoonish, chaotic clicker game inspired by idle games.

## Current Features
- Manual clicking for points
- Buyable characters and pets
- Auto point generation
- Upgrades
- Random chaotic events
- Save and reset support
- Placeholder image slots for future photo wiring

## Deploy
Enable GitHub Pages from the `main` branch root.

## Images
Put your future image files inside the `assets/` folder and update the file paths in `script.js`.
